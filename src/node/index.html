www-lite-server

Leverage a single installation of lite-server for multiple projects.

Description

Install this www-lite-server once and use it to serve multiple projects. Just use the configuration to point it at your existing project.

Configuration

The configuration file is bs-config.json. Edit the server baseDir and set it to the location of index.html in your source.
